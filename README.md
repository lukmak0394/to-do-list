# to-do-list
My first full-stack to-do list application

# Installation 
1. You need a database to store tasks with at least one table "tasks" with 2 columns - task_id & task_name
2. Please also make sure to put credentials required to connect in src/database.php (screenshot below)

# Features
1. Adding new task (creating task in database)
2. Displaying tasks list in DESCENDING order - newest are at the top
3. Filtering tasks (visible on screenshot)
4. Deleting selected task
5. Clearing whole list
